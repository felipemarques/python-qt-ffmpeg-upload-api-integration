# How to use
You need to get your USER_TOKEN from MM Media Manager

# Variables

```
URL_API="https://media-manager.local.mmti.io/api/v1"
USER_TOKEN = "{YOUR USER TOKEN HERE}"
DIRECTORY = "/your path for the videos/obs-studio"
```
